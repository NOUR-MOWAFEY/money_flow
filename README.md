# money_flow

**Money Flow – Expense Tracker**  

Money Flow is a **Flutter mobile application** that helps users track their income and expenses. The app allows users to record financial transactions, store them locally using **Hive**, and easily monitor where their money goes. With a simple and intuitive UI, Money Flow helps users understand and control their spending through charts and organized transaction management.

---

## Features

- Add Income and Expense transactions with date and category  
- Offline storage using Hive (no internet required)  
- Visual insights with **bar charts and line charts** for income and expenses  
- Easy to use & simple UI  
- Helps users control spending & understand money flow  

---

## Tech Stack

- **Flutter** – Frontend framework  
- **Dart** – Programming language  
- **Hive** – Local database for offline storage  

---

## App Flow

1. Open the app → Dashboard/Home  
2. Add a transaction: choose **Income** or **Expense**, enter amount, select category and date  
3. Save → transaction stored locally in Hive  
4. View all transactions and track finances over time with charts
